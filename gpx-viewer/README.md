# GPX Viewer

## Overview
GPX Viewer is a React application that allows users to upload GPX files and visualize the data on a map dashboard. The application provides an intuitive interface for loading and displaying GPX data, making it easy to analyze geographical information.

## Features
- Upload GPX files and parse the data.
- Visualize GPX data on an interactive map.
- User-friendly dashboard for data representation.
- Responsive design for various devices.

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/gpx-viewer.git
   ```
2. Navigate to the project directory:
   ```
   cd gpx-viewer
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application
To start the development server, run:
```
npm start
```
The application will be available at `http://localhost:3000`.

### Building for Production
To create a production build, run:
```
npm run build
```

### Deploying to GitHub Pages
To deploy the application to GitHub Pages, run:
```
npm run deploy
```

## Usage
1. Open the application in your browser.
2. Use the `GpxUploader` component to upload your GPX files.
3. View the visualizations on the `Dashboard` and interact with the `Map`.
